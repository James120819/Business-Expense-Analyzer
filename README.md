# Business Expense Analyzer 

An interactive Streamlit dashboard that analyzes income, expenses, and profit for a small business. This project uses real financial data from my business, Merriness Fortune,
and it demonstrates practical skills in Python, data analysis, and dashboard development. 


# Features 

## Real Time Financial Metrics 
- Total income
- Total Expense
- Net Profit


## Interactive Visualizations
- Expense Breakdown Pie Chart
- Income Over Time Line Chart
- Filterable date ranges


## Clean and Modern Interface
- Sidebar controls
- Organized data tabs
- Auto updating visuals


# Technologies Used

- Python 3
- Streamlit / web UI framework
- Pandas / data analysis
- Matplotlib / visualization


# Project Structure

- app.py
- requirements.txt
- data (Folder)
- expenses.csv
- income.csv


# How to Run the Project

*Use the Terminal tab for the following steps to run this project

1. Install dependencies
   bash:    

   pip install -r requirements.txt

* If you are using python3 use this:

   - python3 -m pip install -r requirements.txt
 
# Run the Streamlit App

streamlit run app.py

# Dashboard Final Steps

* The browser will open automatically, however if it doesn't, you'd need to click on this:

  http://localhost:8501
